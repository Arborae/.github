## ⭐ Progetti in evidenza

### 🐳 d2ha — Docker to Home Assistant
Monitoraggio e controllo dei container Docker con web UI moderna e MQTT autodiscovery per Home Assistant.

→ [Sito & docs](https://arborae.github.io/docker2homeassistant/) · [Repository](https://github.com/arborae/docker2homeassistant)

### 🧩 ha-appliance *(in sviluppo)*
Appliance/stack modulare per semplificare deploy, wizard, App Store e gestione container in modo user-friendly.

→ [Repository](https://github.com/arborae/ha-appliance)

## 🧭 Principi

1. **Documentazione prima di tutto** — se non è spiegato bene, non è finito
2. **Modularità e manutenzione** > “hack veloce”
3. **UX chiara** — se qualcosa confonde, è un bug
4. **Sicurezza e trasparenza** — disclosure responsabile
